Blog Post Using Github

 

1. Create a Public repository with your username.github.io

 

2. You can use any of the git client to perform pull and push ** the repository. it cloud be GitBash, GitHub desktop client or whatever you are familiar with.
I'm using GitBash for my convenience. 

3. Clone the newly created repository with the git clone command.
    a. git clone https://github.com/kunal2791/yourusername.github.io.git
    b. cd yourusername.github.io
    c. echo "hello world" > index.html
    d. push changes to the repository.

4. You are all set to hit and share your post.

    https://yourusername.github.io
